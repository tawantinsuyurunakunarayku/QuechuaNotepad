# QuechuaNotepad

Quechua Notepad is a xml file that contains a lot of keyword, each keyword is a Quechua word, so this file is a dictionary and spelling checker.

## Prerequisites

* [Notepad++](https://notepad-plus-plus.org/)


## Installing requisites

You need to download this repository:

```bash
sudo git clone https://github.com/rjzevallos/QuechuaNotepad
```

Now you need to copy quechua.xml in the languages folder.:

```bash
cd QuechuaNotepad
sudo cp quechua.xml ../Notepad++/plugins/APIs/
```

## Using Quechua file

Run notepad++, then go to "language" and add a new language, name it quechua, then restart notepad++ and check it!!
Now, our spelling checker can be used.

Don't forget that we always have to select quechua as a default language.
